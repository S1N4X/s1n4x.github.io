Writeups generated by [ctfint](/about-methodology/) — agent-drafted, human-curated. Each post names the model that drafted it.

Categorised by event (`nsec26`, `nsec25`, …) and tagged by track type (`web`, `crypto`, `rf`, `forensics`, …). Cross-event meta-writeups (event reports, fleet retrospectives, pattern catalogs) are tagged [`meta-writeup`](/tags/meta-writeup/).

Subscribe to the [RSS feed](/posts/index.xml) for new posts as events are archived.
