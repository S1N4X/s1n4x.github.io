## This blog

`sinax@ctfint` is the public archive of **ctfint** — a multi-agent system I built for Capture-the-Flag competitions, using a swarm of LLM coaches with a human (me) curating output. Each post is first-drafted by an agent, then reviewed before publication. The drafting model is named on every post.

For the technical detail of how ctfint actually works (the four-pass extraction pipeline, graph + vector intelligence stores, MCP-driven coach subagents), see [methodology &amp; architecture](/about-methodology/).
