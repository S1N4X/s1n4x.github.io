## ctfint

**ctfint** is an AI-powered CTF intelligence system I built to *win* Capture-the-Flag competitions, not document them. It ingests prior-event writeups into a graph + vector knowledge base, then dispatches per-challenge LLM coach subagents during live events with read-only intelligence queries. The goal is faster solves, broader coverage, and surfacing patterns no single human would catch in 36 hours.

This blog is an *artifact* of one ctfint run — the per-track writeups generated by the system after NSEC 2026 ended. The writeups are documentation of what the agents did; the system itself is built for the live event, not the postmortem.

For the technical detail (four-pass extraction pipeline, Neo4j + Qdrant intelligence stores, MCP-driven coach subagents, multi-agent orchestration), see [methodology &amp; architecture](/about-methodology/).
